# Phase 1: FMPL CLI and Web REPL

## Goal

Build a minimal FMPL interpreter with CLI and web REPL to enable rapid iteration on the language design.

---

## Deliverables

1. **fmpl-core** - Rust library: lexer, parser, Indexed RPN compiler, VM
2. **fmpl-cli** - Terminal REPL with readline support
3. **fmpl-web** - Axum-based web REPL with htmx

---

pub enum Instruction {
    LoadInt(i64),
    LoadFloat(f64),
    LoadString(String),
    LoadIdent(String),
    LoadNull,
    LoadSelf,

    // Binary ops
    Add,
    Sub,
    Mul,
    Div,
    Eq,
    Lt,
    Gt,
    And,
    Or,

    // Control flow (with target indices)
    JumpIfFalse(usize),
    Jump(usize),

    // Objects
    GetProp(String),
    SetProp(String),
    Call(usize),          // arg count
    MethodCall(String, usize),

    // Binding
    Bind(String),
    BlockStart,
    BlockEnd,

    // Lists/Maps
    MakeList(usize),
    MakeMap(usize),

    // Objects
    Spawn(usize),         // arg count

    Return,
}

pub struct CompiledCode {
    pub instructions: Vec<Instruction>,
}

### Step 5: VM (fmpl/src/vm.rs)

Execute Indexed RPN:

```rust
pub struct VM {
    pub objects: ObjectDB,
    pub call_stack: Vec<Frame>,
}

pub struct Frame {
    pub code: Rc<CompiledCode>,
    pub ip: usize,
    pub values: Vec<Value>,
    pub locals: HashMap<String, Value>,
    pub this: Option<ObjectId>,
}

impl VM {
    pub fn run(&mut self, code: &CompiledCode) -> Result<Value, Error> {
        let mut ip = 0;
        let mut values: Vec<Value> = vec![];

        while ip < code.instructions.len() {
            match &code.instructions[ip] {
                Instruction::LoadInt(n) => {
                    values.push(Value::Int(*n));
                    ip += 1;
                }
                Instruction::Add => {
                    let b = values.pop().unwrap();
                    let a = values.pop().unwrap();
                    values.push(a.add(&b)?);
                    ip += 1;
                }
                Instruction::JumpIfFalse(target) => {
                    let cond = values.pop().unwrap();
                    if cond.is_falsy() {
                        ip = *target;
                    } else {
                        ip += 1;
                    }
                }
                // ... etc
            }
        }

        Ok(values.pop().unwrap_or(Value::Null))
    }
}
```

### Step 6: Object Database (fmpl/src/objects.rs)

In-memory object store:

```rust
pub type ObjectId = u64;

pub struct Object {
    pub id: ObjectId,
    pub parent: Option<ObjectId>,
    pub properties: HashMap<String, Value>,
    pub methods: HashMap<String, CompiledCode>,
}

pub struct ObjectDB {
    pub objects: HashMap<ObjectId, Object>,
    pub next_id: ObjectId,
}

impl ObjectDB {
    pub fn create(&mut self, parent: Option<ObjectId>) -> ObjectId {
        let id = self.next_id;
        self.next_id += 1;
        self.objects.insert(id, Object {
            id,
            parent,
            properties: HashMap::new(),
            methods: HashMap::new(),
        });
        id
    }

    pub fn get_property(&self, id: ObjectId, name: &str) -> Option<Value> {
        let obj = self.objects.get(&id)?;
        if let Some(val) = obj.properties.get(name) {
            return Some(val.clone());
        }
        if let Some(parent) = obj.parent {
            return self.get_property(parent, name);
        }
        None
    }
}
```

### Step 7: CLI REPL (fmpl-cli/src/main.rs)

```rust
use rustyline::{Editor, Result};
use fmpl::{Lexer, Parser, Compiler, VM};

fn main() -> Result<()> {
    let mut rl = Editor::<()>::new()?;
    let mut vm = VM::new();

    println!("FMPL v0.1.0");

    loop {
        match rl.readline("fmpl> ") {
            Ok(line) => {
                rl.add_history_entry(&line);

                match eval(&mut vm, &line) {
                    Ok(value) => println!("=> {}", value),
                    Err(e) => println!("Error: {}", e),
                }
            }
            Err(_) => break,
        }
    }

    Ok(())
}

fn eval(vm: &mut VM, input: &str) -> fmpl::Result<fmpl::Value> {
    let tokens = Lexer::new(input).collect()?;
    let ast = Parser::new(&tokens).parse()?;
    let code = Compiler::new().compile(&ast)?;
    vm.run(&code)
}
```

### Step 8: Web REPL (fmpl-web/src/main.rs)

```rust
use axum::{Router, routing::{get, post}, response::Html, extract::State};
use std::sync::{Arc, Mutex};
use fmpl::VM;

#[tokio::main]
async fn main() {
    let vm = Arc::new(Mutex::new(VM::new()));

    let app = Router::new()
        .route("/", get(index))
        .route("/eval", post(eval))
        .with_state(vm);

    axum::Server::bind(&"0.0.0.0:3000".parse().unwrap())
        .serve(app.into_make_service())
        .await
        .unwrap();
}

async fn index() -> Html<&'static str> {
    Html(include_str!("../static/index.html"))
}

async fn eval(
    State(vm): State<Arc<Mutex<VM>>>,
    body: String,
) -> String {
    let mut vm = vm.lock().unwrap();
    match fmpl::eval(&mut vm, &body) {
        Ok(val) => format!("=> {}", val),
        Err(e) => format!("Error: {}", e),
    }
}
```
